# 目的・シチュエーション

目的：
- MSRPC（ポート135）を通じて、ユーザーやグループ、ドメイン情報などを列挙する。

シチュエーション：
- ポート135もしくは593が開いているWindowsマシン
- NULLセッション or 認証情報あり

---

# MSRPCとは

- Microsoft Remote Procedure Callは、Windows内部機能をリモート実行する仕組み
	- [[用語#msrpc、microsoft-ds, NetBIOS]]

- MSRPCは1つのポート専用ではなく、複数の「通信経路（プロトコル）」を持つ

| プロトコル        | ポート         | ペンテストにおける有用性                      |
| ------------ | ----------- | --------------------------------- |
| ncacn_ip_tcp | 135 + 動的    | 重要                                |
| ncacn_np     | 445/139     | 最重要（[[139,445 -NetBIOS, SMB]]で列挙） |
| ncacn_http   | 593/80/443  | ほぼ不要                              |
| ncadg_ip_udp | 135/udp     | ほぼ不要(outlook等用)                   |
| 動的RPCポート     | 49152–65535 | 完全に不要                             |
|              |             |                                   |

# MSRPCの全体像把握

利用可能なRPCサービスの表示
```zsh
impacket-rpcdump <domain>@<TargetIP> | tee rpcdump.log
```

脆弱性の探索
```sh
nmap -sV -script msrpc-enum -Pn <TargetIP> -p 139
```

# rpcclientによる対話操作

NULLセッション
```zsh
rpcclient $TargetIP -U "" -N
```

認証あり
```zsh
rpcclient $TargetIP -W <DOMAIN> -U <username> -P <pw>
```

## よく使うコマンド（rpcclient接続後）

ユーザー列挙
```zsh
# ユーザーリストの表示
enumdomusers
querydispinfo

# ユーザーの詳細表示
queryuser 0x<RID>

# ユーザー名からSID逆引き
lookupnames <username>
```

グループ列挙
```zsh
# グループリストの表示
enumdomgroups

# グループの詳細表示
querygroup 0x<RID>

# グループのメンバー表示
querygroupmem 0x<RID>
```

ドメイン・SID関連
```zsh
# ドメインリストの表示
enumdomains

# ドメイン情報の表示
querydominfo

# SIDの取得
lsaquery

# SIDの表示
lsaenumsid

# RIC cycling(SIDの探索)
lookupsids [SID]
```

エイリアス(ローカルグループ）の列挙
```zsh
# エイリアスリストの表示
enumalsgroups <builtin|domain>

# エイリアスのメンバー表示
queryaliasmem <builtin|domain> 0x<RID>
```

# ユーザー情報ダンプ

NULLセッション
```zsh
impacket-samrdump -port 139 $TargetIP
```

```zsh
impacket-samrdump -port 139 <domain>/<username>:<pw>@$TargetIP
```
