# 目的・シチュエーション

- 目的
	- MSRPC(SAMR/LSA)を利用し、ローカルSAMやマシン単位の情報を列挙・操作する
	- LDAPでは取得できないローカル管理情報やアカウント操作を補完する

- シチュエーション
	- SMB(445)またはRPC(135/593)が開いているWindowsマシン   
	- NULLセッション or 認証情報あり    
	- LDAPが使えない/補完したい場合

- 位置付け
	- LDAP/BloodHound = メイン（AD構造）    
	- MSRPC = 補助（ローカル/OS操作）

# MSRPCとは

- Microsoft Remote Procedure Callは、Windows内部機能をリモート実行する仕組み
	- [msrpc・microsoft-ds・NetBIOS・mc-nmf](../../Misc/用語.md#msrpc・microsoft-ds・NetBIOS・mc-nmf)

- MSRPCは1つのポート専用ではなく、複数の「通信経路（プロトコル）」を持つ

| プロトコル        | ポート         | ペンテストにおける有用性                                                  |
| ------------ | ----------- | ------------------------------------------------------------- |
| ncacn_ip_tcp | 135 + 動的    | 重要                                                            |
| ncacn_np     | 445/139     | 最重要（[139,445 -NetBIOS, SMB](139,445%20-NetBIOS,%20SMB.md)で列挙） |
| ncacn_http   | 593/80/443  | ほぼ不要                                                          |
| ncadg_ip_udp | 135/udp     | ほぼ不要(outlook等用)                                               |
| 動的RPCポート     | 49152–65535 | 不要                                                            |

# MSRPCの全体像把握

利用可能なRPCサービスの表示
```zsh
impacket-rpcdump <domain>@<target_IP> | tee rpcdump.log
```

脆弱性の探索
```sh
nmap -sV -script msrpc-enum -Pn <target_IP> -p 139
```

# rpcclientによる対話操作

NULLセッション
```zsh
rpcclient <target_IP> -U "" -N
```

認証あり
```zsh
rpcclient //<target_IP> -U <domain>/<username>
```

## よく使うコマンド（rpcclient接続後）

ユーザー列挙
```zsh
# ユーザーリストの表示
enumdomusers
querydispinfo

# ユーザーの詳細表示
queryuser 0x<RID>

# ユーザー名からSID逆引き
lookupnames <username>
```
- `result was NT_STATUS_IO_TIMEOUT`：サーバー側がSAMR (Security Account Manager Remote)での列挙を禁止しており、応答を返さない（結果、DISCONNECTEDになる）

グループ列挙
```zsh
# グループリストの表示
enumdomgroups

# グループの詳細表示
querygroup 0x<RID>

# グループのメンバー表示
querygroupmem 0x<RID>
```

ドメイン・SID関連
```zsh
# ドメインリストの表示
enumdomains

# ドメイン情報の表示
querydominfo

# SIDの取得
lsaquery

# SIDの表示
lsaenumsid

# RIC cycling(SIDの探索)
lookupsids <SID>
```

エイリアス(ローカルグループ）の列挙
```zsh
# エイリアスリストの表示
enumalsgroups <builtin|domain>

# エイリアスのメンバー表示
queryaliasmem <builtin|domain> 0x<RID>
```
