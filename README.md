# OSCP+チートシート

 本リポジトリは、OSCP+（PEN-200）学習を軸に、ペネトレーションテストの 技術・手法をまとめた**日本語チートシート・学習ノート集**です。 試験対策だけでなく、実際のペネトレーションテスト業務でも 参照できるよう、コマンドや手順を体系的に整理しています。

---

# 対象読者

- OSCP+・OSEP取得を目指している方
- ペネトレーションテストの技術を体系的に学びたい方
- 日本語でオフェンシブセキュリティを学びたい方
