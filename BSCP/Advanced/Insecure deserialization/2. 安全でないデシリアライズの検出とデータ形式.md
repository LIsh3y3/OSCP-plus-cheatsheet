## プログラミング言語ごとのシリアライズデータ形式と兆候

### PHP

- データ型を文字で表し、各エントリの長さを数字で表す。

###### ↓オブジェクトの例
```php
$user->name = "carlos";
$user->isLoggedIn = true;
```
- ↓シリアライズされると...
```php
O:4:"User":2:{s:4:"name":s:6:"carlos"; s:10:"isLoggedIn":b:1;}
```
- `0:4:"User:"`: 4文字のクラス名"User"を持つオブジェクト
- `2:`: オブジェクトが2つの属性を持つことを示す
- `s:4:"name":`: 最初の属性のキーは4文字の"name"である
- `s:6:"carlos";`: 最初の属性のバリューは6文字の"carlos"である
- `s:10:"isLoggedIn":`: 2番目の属性のキー(ry
- `b:1;`: booleanのバリューは1=true

###### phpのシリアライズ後の形式

- [データ型表記について詳細](https://www.wakuwakubank.com/posts/423-php-serialize/#:~:text=unserialize%E9%96%A2%E6%95%B0-,serialize%E9%96%A2%E6%95%B0,-echo%20serialize()
- INT型：　`i:100;`
- float型(小数点) : `d:1.0000000...;`
- NULL: `N;`
- Object: `O:lenth:"ObjectName":プロパティの数(サイズ)`
- ARRAY型：
```php
 'array' => array('鈴木', '山田', '佐藤')
```
	↓
```php
a:3:{i:0;s:6:"鈴木";i:1;s:6:"山田";i:2;s:6:"佐藤";}
```
- ⚠️：`s:6`の6はバイト数。漢字は1文字で3バイト
- `i:0`: 配列のキーが整数のインデックスだから。

#### PHPでデシリアライズされている兆候(ソース見れるなら)

- ソースコードに`unserialize()`が含まれているか？
	- `serialize()`でシリアライズし、`unserialize()`でデシリアライズする。

#### Javaでデシリアライズされている兆候(ソース見れるなら)

- ソースコードに`readObject()`メソッドは含まれているか？
	- これは`InputStream`で渡されたシリアライズデータを読み取るメソッド

- どのクラスでも、`java.io.Serializable`をインプリメントしているなら、シリアライゼーション可能
```java
import java.io.Serializable
...

public class ExampleClass implements Serializable {
...
}
```
