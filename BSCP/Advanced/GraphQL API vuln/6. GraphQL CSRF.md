###### GraphQL CSRFが起きるシチュエーションは？

- `Content-Type`ヘッダを検証しないとき
- かつ、CSRF tokenが実装されていないとき
- 以下のリクエストのいずれかが可能なとき

| リクエストメソッド | Content-Type |
| ---- | ---- |
| GET | ヘッダなし |
| POST | `application/x-www-form-urlencoded` |
